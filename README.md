# go-o3o

golang implementation of [o3o](https://github.com/turingou/o3o)


## Installation

`go get github.com/mozillazg/go-o3o/cmd/o3o`


## Usage

    # list all available tags
    $ o3o
     (￣┏Д┓￣ )

    # seach emoticons by tag
    $ o3o happy
     ヾ(･ω･*)ﾉ

    # list all available tags
    $ o3o list
    握手
    xiong
    old
    哈哈
    what
    ...
